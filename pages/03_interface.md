# 🖥️ CARTA Viewer Interface Guide

The **CARTA Viewer** provides a modern, flexible, and highly interactive interface for exploring astronomical images and data cubes. Its layout is designed to maximize usability while giving access to powerful analysis tools.

---

## 🧭 Overview of the Interface

The CARTA interface is composed of several key panels and tools:

- **Image Viewer (Main Canvas)**
- **Toolbar**
- **Layer & File List Panel**
- **Region & Annotation Panel**
- **Spectral & Statistics Widgets**
- **Preferences and Settings**

These components can be rearranged and customized to suit different workflows.

---

## 🖼️ Image Viewer (Main Canvas)

The central area of CARTA is the **image viewer**, where data is displayed.

### Features
- Pan and zoom with mouse or trackpad  
- Multi-resolution rendering for smooth navigation  
- Overlay of contours, grids, and coordinate systems  
- Support for multiple images and linked views  

### Interaction
- **Scroll / pinch** → zoom in/out  
- **Click + drag** → pan  
- **Hover** → inspect pixel values and coordinates  

---

## 🧰 Toolbar

The toolbar provides quick access to common tools and actions.

### Common Tools
- 🔍 Zoom controls  
- ✋ Pan tool  
- 📐 Region drawing tools (rectangle, ellipse, polygon)  
- 🎨 Colormap and scaling controls  
- ⚙️ Settings and preferences  

The toolbar is designed for rapid switching between interaction modes.

---

## 📂 Layer & File List Panel

This panel manages loaded datasets and image layers.

### Capabilities
- Load and unload files (e.g., FITS images)  
- Toggle visibility of layers  
- Adjust rendering order  
- Apply contour overlays  

This is especially useful when comparing multiple datasets.

---

## 📍 Region & Annotation Panel

Regions allow users to define areas of interest for analysis.

### Features
- Create geometric regions (circle, rectangle, polygon)  
- Edit and move regions interactively  
- Label and organize annotations  
- Save and reload region definitions  

Regions are tightly integrated with spectral and statistical tools.

---

## 📊 Spectral & Statistics Widgets

CARTA provides interactive analysis panels linked to the image viewer.

### Spectral Profile
- Displays intensity as a function of frequency/velocity  
- Updates dynamically based on selected region  

### Statistics Panel
- Computes properties such as:
  - Mean  
  - RMS  
  - Sum  
  - Min/Max  

### Key Advantage
All plots update in **real time** as regions or slices change.

---

## 🎛️ Display & Rendering Controls

Users can customize how images are displayed.

### Options
- Colormap selection  
- Linear / logarithmic / custom scaling  
- Contrast and brightness adjustments  
- Contour levels and styling  

These controls allow detailed inspection of faint or complex structures.

---

## 🧩 Multi-Panel & Linked Views

CARTA supports multiple synchronized views.

### Use Cases
- Compare different wavelengths  
- Inspect different channels of a data cube  
- Link zoom and cursor position across panels  

This is particularly useful for multi-dimensional datasets.

---

## ⚙️ Preferences & Settings

The settings menu allows customization of the interface.

### Examples
- Theme (light/dark mode)  
- Performance options  
- Coordinate display formats  
- Default visualization settings  

---

## 💡 Key Strengths of the Interface

- **Responsive and interactive** even with large datasets  
- **Highly customizable layout**  
- **Real-time feedback** for analysis tools  
- **Seamless integration** of visualization and quantitative analysis  

---

## 📌 Summary

The CARTA interface combines powerful visualization with interactive analysis tools in a clean, browser-based environment. Its modular design allows users to adapt the layout to their needs while maintaining fast and efficient workflows.

Whether exploring a single image or analyzing complex spectral cubes, CARTA provides an intuitive and scalable interface for modern astronomy.
